Light\+\_\+controller constructor. 

This constructor initializes the mode, brightness level and the color the LEDstrips should have. The mode should be an integer 1, 2 or 3. The brightness should be an 8 bit integer in the range 0 and 31. The colors should be an 8 bit integer in the range of 0 and 255. Only the \textquotesingle{}mode\textquotesingle{} will be directly added to the private. The brightness and the colors will first be converted to arrays.
