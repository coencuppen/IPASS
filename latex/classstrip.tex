{\footnotesize\ttfamily void strip\+::led\+Strip (\begin{DoxyParamCaption}\item[{int}]{leds }\end{DoxyParamCaption})}



led\+Strip function 

This function manages with the parameter \textquotesingle{}leds\textquotesingle{} which LED\textquotesingle{}s should be on. It also manages the color of the LED\textquotesingle{}s. Firs the function turns all the LED\textquotesingle{}s of with the \mbox{\hyperlink{classstrip_a4f9cd0d264ef1e1e246e1c3c90369968}{led\+Off()}} function. The function reads the mode that has been given to the \mbox{\hyperlink{classlight_controller}{light\+Controller}} class. If the mode is 1 the first 5 LED\textquotesingle{}s are green, the 6 and 7th LED are yellow and the 8th LED is red. If the mode is 2 the first 5 LED\textquotesingle{}s are green, the 6 and 7th LED are yellow and changes the first 5 LEDS to yellow. The 8th LED turns all LED\textquotesingle{}s to red.

The apa102 LED strip has a protocol to write the amount of LED\textquotesingle{}s and the color. First the function writes 4 bytes whith only 0\textquotesingle{}s, so the LED strip will listen. Then it writes the brightness byte and the colors in the order blue green red. It repeat the last stap for the amount of LED\textquotesingle{}s it has to turn on. When it\textquotesingle{}s done it writes a byte of only 1\textquotesingle{}s so the LED strip stops listening. \mbox{\Hypertarget{classstrip_aa1e03c218989d43d6d8b53fe58f19cb9}\label{classstrip_aa1e03c218989d43d6d8b53fe58f19cb9}}
