int * MSGEQ7::getAmplitude ( int  devider)

getAmplitude function

This function returns an array of integers from 0 to 8. The first integer is the first frequency band. The values are divided by the 'devider' parameter (usually 500) so the max readings are around 8 and the lowest readings are 0.
